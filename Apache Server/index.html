This page was created using Apache with and AWS EC2 Instance on Centos 7.

How did you create this page?

I first installed "httpd" on my EC2 instance running Centos 7 and created the index.html file in the Document Root directory. I edited the file in html to create this page! I changed permissions for the index.html file to Apache for both as is best practice. I then enabled and started the httpd service and was good to go!
